# ArduPilot LUA

A Collection of LUA Scripts for ArduPilot

- CRSF Menus
    - crsf-calibrate.lua - run calibration functions from your transmitter
    - flight.lua - change aspects of your flight configuration
    - gain_tuner.lua - easily modify your tune and logging setup
    - modules/crsf_helper.lua - library to support CRSF menus

- update_scripts.py - easily upload a directory of scripts to your autopilot
